本目录为业务服务层：承接 controller 调用，聚合数据库读写、模板资源扫描与推荐排序、收藏/使用量统计、构图模型推理封装，以及转发到 Python VLM 服务等；这里集中放业务规则与容错。user_service.go：注册/登录/改名改密/头像（bcrypt+唯一键处理）；material_service.go：素材增查删、草稿转作品、文件清理；composition_service.go：调用 composition 推理并返回构图结果；template_service.go：读取 templates.json 与静态目录生成模板列表/详情；template_recommend_service.go：热门/偏好/收藏/使用信号推荐与搜索融合；template_favorite_service.go：收藏增删查（幂等）；template_usage_service.go：使用量自增与查询；preference_service.go：偏好 tags upsert；vlm_service.go：multipart 转发到 VLM_API_URL、超时与 JSON 校验。readme.md：目录说明。
