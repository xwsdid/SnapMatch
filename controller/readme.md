本目录为 Gin HTTP 控制层：负责路由入参解析、基础校验、统一返回结构与错误码封装，并调用 service 完成业务；尽量保持“薄”，不直接承载复杂逻辑/SQL。user_controller.go：注册、登录、查询“我”、改名改密、头像上传；material_upload_controller.go：素材上传到 static/uploads 并落库（可带 template_id/使用量）；photo_controller.go：素材列表、删除（软删+文件清理）、草稿转作品；sync_controller.go：草稿同步（user_id+url 入库）；composition_controller.go：构图分析入口；vlm_controller.go：转发图片+task 到 Python VLM；template_controller.go：模板热门/列表/推荐/详情/搜索；template_favorite_controller.go：收藏/取消/列表；preference_controller.go：偏好 tags 读写；readme.md：目录说明。
